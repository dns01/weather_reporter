### 介绍

本程序是一个命令行程序，用来获取指定城市的天气信息。

可以使用以下命令构建程序：

```go
go build -o weather-reporter .
```

如何使用本程序？

```azure
./weather-reporter {cityname}

未指定城市，使用默认城市：郑州
```
